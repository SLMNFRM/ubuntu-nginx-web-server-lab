# Ubuntu Nginx Web Server Lab

## Project Overview

This lab demonstrates how to deploy a Linux web server using Ubuntu Server and Nginx inside a virtual machine. The environment simulates a basic server deployment used in enterprise environments.

The project was built using VMware Workstation and documents the full installation, configuration, and validation process.

## Lab Objectives

- Deploy a Linux server in a virtual environment
- Install and configure a web server
- Host a custom HTML webpage
- Verify network connectivity to the server
- Document the deployment process
